## 统计非回文子串数量

### 问题描述

给定一个长度为 n 的字符串 s（仅包含小写字母）。定义“包裹字符串”为：子串的首字符与尾字符相同。现对于字符串 s 的每一个前缀 s[1…i]（1 ≤ i ≤ n），统计该前缀的所有子串中，不是包裹字符串的子串数量。

输入：第一行输入整数 n（1 ≤ n ≤ 2×10⁵），表示字符串长度；第二行输入长度为 n 的字符串 s。

输出：共输出 n 行。第 i 行输出一个整数，表示前缀 s[1…i] 的所有子串中，不是包裹字符串的子串数量。

- 样例：

  - 输入：

    ```
    4
    abca
    ```

  - 输出：

    ```
    0
    1
    3
    5
    ```

### 思路

非包裹字符串 = 总字符串 - 包裹字符串

对于前缀 [1..i]，总字串数 = `i * (i + 1) / 2`，所以当总字符串长度+1时，总字符串增量是 `i + 1`

对于每个字符 `c`，假设在 [1..i] 中出现了 `k` 次，那么包裹字符串的数量就是 `k * (k + 1) /2` （注意单个字符也算包裹字符串）。所以字符串中增加一个 c 时，包裹字符串增量是 `k + 1`

因此，当进行下一次输出时，不需要重新遍历字符串，而是统计增量，也就是 `i - k`

### 代码

```Java
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        String str = sc.next();

        long total = 0;     // 总字符串数量
        long wrap = 0;      // 包裹字符串数量
        long[] duplicates = new long[26];   // 统计重复字母数量

        for (int i = 0; i < n; i++) {
            total += i + 1;             // 总字符串数量的增量是 i + 1
            wrap += duplicates[i] + 1;  // 包裹字符串的数量是 k + 1
            duplicates[str.charAt(i) - 'a']++;
            System.out.println(total - wrap);
        }
    }
}
```
